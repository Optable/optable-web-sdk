Example index.html importing a locally built main.js

The index.html you are rendering is importing a locally build main.js (using webpack).

The main.js file imports Optable's web-sdk and makes example API calls to the configured DCN. The SDK was installed using NPM.

To change the compiled main.js, edit the file demos/npm/src/identifyAndTargeting.js.
Make sure to configure the SDK according to your DCN's host and use the slug of the site you created in the DCN.
Then build again using webpack from the folder demos/npm: 
      npm run build
Finally, refresh this page (or open the generated index.html file located in demos/npm/dist/index.html).

Each render will execute the generated code located in demos/npm/dist/main.js and output logs in the web console.
